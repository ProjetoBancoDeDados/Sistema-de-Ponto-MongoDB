Sistema de Controle de Ponto
Este projeto é um sistema de controle de ponto que permite gerenciar funcionários, registrar entradas e saídas, e gerar relatórios de horas trabalhadas. O sistema é implementado em Python e utiliza um banco de dados MongoDB para armazenamento dos dados.

Funcionalidades
Gerenciamento de Funcionários:

Inserir novos funcionários.
Atualizar informações de funcionários existentes.
Remover funcionários.
Registro de Ponto:

Inserir registros de entrada e saída.
Atualizar registros de ponto.
Remover registros de ponto.
Relatórios:

Gerar relatórios de horas trabalhadas por funcionário.
Gerar relatórios de registros de ponto.
Requisitos
Python 3.x
MongoDB
Bibliotecas:
pymongo
Instalação
Instale as dependências necessárias:


pip install pymongo
Certifique-se de que o MongoDB está instalado e em execução. Você pode iniciar o MongoDB com o seguinte comando:


mongod
Uso
Abra o terminal e execute o script principal:


python main.py
Siga as instruções exibidas no terminal para navegar pelo sistema.

Estrutura do Código
models/: Contém as classes que representam os modelos de dados (Funcionário, Registro de Ponto).
reports/: Contém as classes responsáveis pela geração de relatórios.
main.py: O ponto de entrada do sistema, onde o controlador é instanciado e executado.
